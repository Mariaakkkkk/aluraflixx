Olá! Neste vídeo, vamos dar início ao nosso projeto.

Criando o arquivo HTML

O projeto em que vamos trabalhar será uma página que colocaremos na internet, contendo uma coleção de vídeos sobre assuntos que gostamos. No caso do instrutor, ele escolherá vídeos sobre filmes, séries e animações, mas você pode escolher vídeos sobre qualquer assunto do seu interesse.

Como trabalharemos com filmes da Alura, vamos nomear o projeto como AluraFlix. Para criar esse projeto, começaremos criando uma pasta no computador onde armazenaremos os arquivos.

Criando a pasta do projeto

Inicialmente, abrimos a pasta principal onde vamos criar a pasta do projeto, depois clicamos com o botão direito e escolhemos a opção "Novo > Pasta". No caso do instrutor, a pasta se chamará "aluraflix". Após pressionar "Enter", teremos a pasta do projeto criada.

Abrindo a pasta do projeto no VS Code

O próximo passo é começar a escrever arquivos com código que irão descrever a página para que as pessoas possam acessá-la. Para fazer isso, precisamos de algum programa de edição de código.

Existem diversos programas, mas utilizaremos o Visual Studio Code (ou VS Code). Caso ainda não tenha ele instalado na sua máquina, explicamos o processo de instalação nas atividades desta aula.

Uma vez instalado o programa, vamos acessá-lo. A partir da tela inicial, podemos utilizar o menu lateral esquerdo para abrir uma pasta, ou seguir o caminho "File > Open Folder…".

Por padrão, o VS Code vem no idioma inglês.

Feito isso, vamos selecionar a pasta do projeto ("aluraflix"), que por enquanto está vazia. Após clicar em "Selecionar pasta", o VS Code pergunta se confiamos na pessoa que criou a pasta ("Do you trust the authors of the files in this folder?"). Basta clicar no botão azul "Yes, I trust the authors" ("Sim, eu confio nos autores") na parte inferior.

Criando o arquivo index.html

Agora estamos prontos para criar a página. Para isso, precisamos criar um arquivo, então no menu lateral esquerdo, clicaremos com o botão direito e selecionaremos a opção "New File…".

Quando criamos páginas na internet, a primeira página sempre deve ser criada com o mesmo nome fixo, que é o nome do índice, pois será a página principal de onde partimos para conhecer o restante do site. "Índice" em inglês é "Index", então vamos criar um arquivo chamado index.html.

A extensão .html é o padrão de uma página na internet.

Após criar o arquivo, vamos digitar o que queremos na página. Inicialmente, queremos um título, que nesse caso será "ALURAFLIX". Feito isso, vamos salvar em "File > Save" (atalho "Ctrl + S").

index.html:

ALURAFLIX

Em seguida, vamos abrir esse arquivo no navegador. Para isso, clicamos com o botão direito sobre o arquivo index.html. Dessa forma, encontraremos várias opções de como abrir esse arquivo.

Nesse caso, vamos escolher a opção "Reveal in File Explorer" para revelar no explorador. Ao fazer isso, o arquivo index.html será exibido na pasta do projeto "aluraflix".

Com um duplo clique nesse arquivo, ele será aberto no navegador. Nesse momento, teremos uma página apenas com a palavra "ALURAFLIX" no canto superior esquerdo. Já temos o começo da página, apenas com uma palavra. Porém, queremos muito mais, então vamos dar continuidade.

Criando o código HTML

De volta ao Visual Studio Code, vamos trabalhar com o código para criar uma página mais complexa e rica. Primeiramente, vamos pegar as informações do vídeo que queremos colocar na página.

O primeiro vídeo que iremos colocar será de um filme sobre o Aranhaverso. Com o YouTube aberto no navegador, vamos acessar o vídeo "ATRAVÉS DO ARANHAVERSO SUPERA O PRIMEIRO FILME?".

Queremos usar esse título, então vamos selecionar o texto e copiar (atalho "Ctrl + C"). Feito isso, vamos voltar para o Visual Studio Code, teclar "Enter" algumas vezes para adicionar linhas em branco na página, e colar com o atalho "Ctrl + V" (ou clicando com o botão direito e selecionando "Paste").

Em seguida, na linha abaixo, vamos adicionar uma tag, por exemplo, "#homem-aranha".

ALURAFLIX


ATRAVÉS DO ARANHAVERSO SUPERA O PRIMEIRO FILME?
#homem-aranha

Após salvar, podemos voltar ao navegador e atualizar a página (atalho "Ctrl + R"). Como resultado, teremos o texto "ALURAFLIX ATRAVÉS DO ARANHAVERSO SUPERA O PRIMEIRO FILME? #homem-aranha" em uma única linha. O navegador entende o documento como um texto corrido, e não é isso que queremos.

Analisando o código no momento, encontramos vários textos soltos. Por padrão, quando o navegador lê esse texto solto, ele assume algumas coisas. Por exemplo: ele assume que os espaços que adicionamos com "Enter" correspondem a um único espaço em branco, não linhas em branco.

Existem regras muito bem definidas de como devemos escrever uma página, da mesma forma que existem regras para escrever uma redação, ou uma fórmula matemática, por exemplo. No caso da fórmula matemática, precisamos colocar os símbolos em ordens específicas para ela funcionar corretamente.

Sendo assim, devemos ter cuidado e indicar para o navegador o que é cada coisa. O que é o título na página? O que não é o título? O que está em uma linha? O que está em outra?

Definindo o cabeçalho da página

O texto "ALURAFLIX" no topo do código corresponde ao cabeçalho da página. Se é o cabeçalho, vamos escrever a palavra "cabeçalho", mas através de um código, assim como na matemática.

Para iniciar esse código, utilizamos o símbolo de menor (<) seguido da palavra "cabeçalho" em inglês, ou seja, header, e do símbolo de maior (>) para fechar. Assim, teremos a tag <header>. Quando fechamos com o símbolo >, automaticamente, o VS Code fecha a tag com </header>.

Tudo o que estiver entre esses dois códigos será o cabeçalho da página. Dito isso, vamos recortar (atalho "Ctrl + X") a palavra "ALURAFLIX" e colar entre <header> e </header>. 

<header>ALURAFLIX</header>


ATRAVÉS DO ARANHAVERSO SUPERA O PRIMEIRO FILME?
#homem-aranha

Definindo o título da página

Além disso, queremos que a informação da linha 4, ou seja, o texto "ATRAVÉS DO ARANHAVERSO SUPERA O PRIMEIRO FILME?", seja um título importante. Para adicionar títulos, usamos outra tag.

Começaremos abrindo novamente uma tag com <, a qual receberá a letra h e o nível de importância. O número 1 é o mais importante, então adicionaremos h1 seguido do símbolo > (<h1>). Automaticamente, o VS Code irá fechar a tag com </h1> logo em seguida.

Entre essas duas tags, podemos colocar o texto desejado como título, então vamos recortar todo o título do vídeo e colar entre <h1> e </h1>.

<header>ALURAFLIX</header>


<h1>ATRAVÉS DO ARANHAVERSO SUPERA O PRIMEIRO FILME?</h1>
#homem-aranha

Vamos conferir o resultado até o momento? Ao atualizar a página no navegador, teremos um cabeçalho na parte superior contendo o texto "ALURAFLIX", seguido de um título grande na linha abaixo ("ATRAVÉS DO ARANHAVERSO SUPERA O PRIMEIRO FILME?"), e de um texto solto na última linha ("#homem-aranha").

Definindo o parágrafo da página

O texto "#homem-aranha" ainda está solto no código, pois não utilizamos nenhuma tag. Nesse caso, queremos criar um parágrafo de texto, e também existe uma tag específica para isso.

A tag utilizada para parágrafo é <p>, referente à palavra "paragraph" ("parágrafo" em inglês). Ao adicioná-la, automaticamente, teremos o fechamento da tag com </p>. Entre <p> e </p>, podemos adicionar o texto livre desejado, que neste exemplo é "#homem-aranha".

<header>ALURAFLIX</header>


<h1>ATRAVÉS DO ARANHAVERSO SUPERA O PRIMEIRO FILME?</h1>
<p>#homem-aranha</p>

De volta ao navegador, após atualizar novamente a página, teremos o cabeçalho (<header>), o título principal (<h1>), e o parágrafo (<p>), todos com os textos desejados.

Adicionando o vídeo na página

Nesse momento, temos a primeira versão da página, mas ainda falta algo muito importante: o vídeo. Para começar, vamos acessar a página do vídeo no YouTube.

Logo abaixo do vídeo, encontramos o botão de compartilhar ("Share"). Ao clicar nele, descobrimos várias formas de compartilhamento: no WhatsApp, no Facebook, no e-mail, entre outras.

Como queremos colocar o vídeo dentro da nossa página, o objetivo é embutir. A opção para isso se chama "Embed", representada pelo símbolo da tag que acabamos de conhecer (<>).

Após clicar nessa opção, teremos acesso a um código grande à direita. Vamos selecioná-lo e clicar no botão "Copy" no canto inferior direito para copiar.

Uma vez copiado o código, precisamos colar em algum lugar. De volta ao código no Visual Studio Code, vamos colar após a tag <p> da linha 5.

Lembre-se: no HTML, não importa a quantidade de espaços entre uma linha e outra.

<header>ALURAFLIX</header>


<h1>ATRAVÉS DO ARANHAVERSO SUPERA O PRIMEIRO FILME?</h1>
<p>#homem-aranha</p>

Com isso, temos o arquivo completo. Podemos salvar com "Ctrl + S", voltar para o navegador, e atualizar a página com "Ctrl + R". Como resultado, temos o cabeçalho, o título, a parágrafo, e o vídeo. Se apertarmos o botão de play no vídeo, ele será reproduzido na nossa própria página.

Definindo o corpo da página

Assim, temos um arquivo que é a nossa página. Conhecemos o que acontece nos bastidores da internet, ou seja, como os sites que utilizamos mostram tudo na nossa tela: através de pequenas tags.

Existem inúmeras regras que vamos aprender ao longo das aulas. À medida que aplicamos essas regras, conseguimos adicionar textos, imagens, reproduzir um vídeo, entre várias outras coisas.

Como todo o conteúdo de código que construímos é a página HTML, vamos colocar isso dentro do corpo da página. Para as tags <header>, <h1>, <p> e <iframe> não ficarem soltas na página, vamos utilizar uma tag chamada <body> ("corpo" em inglês) na primeira linha do código. Para fechar, adicionamos a tag </body> ao final do arquivo, na última linha. 

<body>

<header>ALURAFLIX</header>


<h1>ATRAVÉS DO ARANHAVERSO SUPERA O PRIMEIRO FILME?</h1>
<p>#homem-aranha</p>



<iframe width="560" height="315" src="https://www.youtube.com/embed/gt_fAE1Eg2Q?si=c93nZtO1DTNYueO4" title="YouTube video player" frameborder="0" allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture; web-share" referrerpolicy="strict-origin-when-cross-origin" allowfullscreen></iframe>



<body>

Após salvar, não haverá nenhuma diferença no navegador, mas isso é importante para que a página fique bem escrita. Todo o conteúdo está dentro do corpo, e é dessa forma que definimos a página HTML.

Conclusão

Nas próximas aulas, faremos coisas ainda mais legais: vamos adicionar mais vídeos, estilizar com cores, formatos, posições, e assim por diante. Aprenderemos tudo à medida que explorarmos mais sobre páginas HTML e criação de páginas. Até o próximo vídeo!
